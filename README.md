A Comparative Study of Path Planning Algorithms for Indoor Mobile Robots Using Simulation
Overview

This project presents a comparative investigation of classical path-planning algorithms for indoor robotic navigation.

Algorithms evaluated:

Breadth-First Search (BFS)
Dijkstra's Algorithm
A* Search

The study investigates:

Path optimality
Computational effort
Obstacle density effects
Statistical behavior across randomized environments
